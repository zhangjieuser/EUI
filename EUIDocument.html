产品介绍

EUI是一套国内开发并且开源的 Web UI 解决方案 ，采用自身经典的模块化规范，并遵循原生 HTML/CSS/JS 的开发方式，极易上手，拿来即用。其风格简约轻盈，而组件优雅丰盈，从源代码到使用方法的每一处细节都经过精心雕琢，非常适合网页界面的快速开发。准确地说，它更多是面向后端开发者，你无需涉足前端的各种工具，只需面对浏览器本身，让一切你所 需要的元素与交互，从这里信手拈来。

兼容性和面向场景

EUI 兼容人类正在使用的全部浏览器（IE6/7除外），可作为 Web 界面速成开发方案。
